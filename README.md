# 🛒 E-Commerce Web Application

A complete e-commerce web platform built with **PHP, Laravel, MySQL, HTML, CSS, and jQuery**.  
It provides customers with a seamless shopping experience and gives admins full control through a powerful dashboard.  

---

## 📖 About the Project  

This **E-Commerce Web Application** is designed to provide both customers and admins with a smooth and interactive experience.  

- **For Customers:** They can browse products, view details, add items to the cart, manage wishlists, apply coupons, and securely place orders.  
- **For Admins:** The dashboard allows complete control over users, roles, permissions, products, categories, orders, blogs, sliders, and custom blocks.  

The system is fully responsive, optimized for speed, and designed with modern UI/UX principles.  

---

## ⚙️ Key Features  

### 🛍️ Customer Side (Frontend)  
- Browse and search products with categories and filters  
- Product details page with images, size/color options, and stock availability  
- Shopping cart with quantity update and coupon support  
- Wishlist management for favorite products  
- Checkout process with order summary and order tracking  
- Blog section for web-related information and product updates  

### 🛠️ Admin Side (Dashboard)  
- **User Management** – Add, update, or deactivate users  
- **Role & Permission Control** – Secure access management with RBAC  
- **Product Management** – CRUD operations for products and categories  
- **Order Management** – View, track, and manage customer orders  
- **Slider & Blocks Management** – Manage homepage banners and dynamic content  
- **Reports & Analytics** – Sales insights and order tracking  

---

## 🛡️ Why This Project?  
- Provides an **end-to-end solution** for an online shopping system  
- Implements **role-based access control (RBAC)** for security  
- Uses a **modular and scalable structure** for future enhancements  
- Suitable as a **real-world portfolio project**  

---

## 🖥️ Tech Stack  
- **Backend:** PHP, Laravel  
- **Frontend:** HTML5, CSS3, jQuery, Bootstrap  
- **Database:** MySQL  
- **Other Tools:** Yajra DataTables, Git, Composer
